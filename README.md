# Connect Quest

# Tech Stack
Frontend
- React + TypeScript

Backend
- NodeJS + JavaScript
- MongoDB

## How to run the App

### Run the Web App in development mode
``` bash
$ cd app

$ npm install

$ npm run dev
```

### Run the backend server in development mode
``` bash
$ cd backend

$ npm install

$ npm run dev
```
